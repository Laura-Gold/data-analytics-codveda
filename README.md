# data-analytics-codveda

**Data Analytics Portfolio – Codveda Tasks**

**#Overview**

This repository contains my completed data analytics tasks across three levels, covering exploratory data analysis, advanced analytics, and machine learning with dashboard visualization.

# Project Structure

🔹 Level 1: Basic Analysis

* Exploratory Data Analysis (Churn Prediction Dataset)
* Basic Data Visualization  (Churn Prediction Dataset)

🔹 Level 2: Intermediate Analysis

* Time Series Analysis (Stock Prices Dataset)
* Clustering Analysis: K-Means (Churn Prediction Dataset)

🔹 Level 3: Advanced Analysis

* Predictive Modeling (Churn Prediction Dataset) including:

  * Logistic Regression
  * Decision Tree
  * Random Forest (with hyperparameter tuning)
  * Feature Importance Analysis
* Power BI Dashboard with interactive visualizations (Churn Prediction Dataset)

# Dashboard Highlights

* KPI Cards (Churn Rate, Total Customers, High-Risk Customers)
* Churn Drivers Analysis
* Customer Segmentation Insights
* Interactive Filters (Contract Type, Risk Segment, etc.)

# Key Insights

* Customer churn is strongly influenced by pricing, service experience, and usage behavior
* High-risk customers can be identified and targeted proactively
* Data-driven strategies can significantly improve customer retention

# Tools & Technologies

* Python (Pandas, Scikit-learn, Seaborn, Matplotlib)
* Power BI
* Jupyter Notebook
